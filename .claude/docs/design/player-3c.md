# 角色3C（Character / Camera / Control）

## 角色数值

### 生命值与护甲

- **最大生命值（设计值）**：6点（方便做倍数运算）
- **显示方式**：6颗心，每颗心代表2点生命值
  - 每颗心有三种状态：0点、1点、2点血量 → 对应三张素材
- **最大护甲值**：与最大生命值一致，每颗心上最多存储2点护甲
  - 1点、2点护甲 → 对应两张素材
- **自然恢复**：持续5秒未受伤害后开始恢复护甲
- **护甲恢复速度**：2秒回复1点，直到回满
- **生命值**：不会自然恢复

## 移动能力

### 基本移动

- **八向移动**（WASD），方向改变时维持当前速度和加速度，仅改变方向
- **按下开始移动，松开立刻停止**（速度立即重置为零）
- 若有冰面/低摩擦力地面，另外设定参数
- **移动速度**：5 m/s

### 跳跃

随按键时间长短决定跳跃高度：

- **短按**：在时间点 t₁ 之前松开 → 高度 h₁
- **长按**：在时间点 t₂ 之后松开 → 高度 h₂
- **中间区域**（t₁ 到 t₂ 之间松开）：
  $$h_t = h_1 + (h_2 - h_1) \times \sqrt{\frac{t - t_1}{t_2 - t_1}}$$

> 参数 t₁、t₂、h₁、h₂ 待策划确定具体数值。**最大跳跃高度**：3m。

#### 跳跃方案一（瞬时速度 + 匀速 + 减速）

**短按**（在 t₁ 之前松开）：
- 按下时垂直速度立刻变为 v₁
- 维持最大速度至 t₁
- t₁ 时立即垂直速度清零，按下坠加速度开始下落
- $$v_1 = \frac{h_1}{t_1}$$

**长按**（超过 t₂ 松开）：
- 超过 t₁ 后未松开 → 以固定加速度 a₂ 减速
- 检测到松开后 → 立即垂直速度清零 → 按下坠加速度开始下落
- $$a_2 = \frac{h_1^2(t_2 - t_1)}{2t_1^2(h_2 - h_1)}$$

#### 跳跃方案二（匀加速 + 匀速 + 匀减速）

**短按**（在 t₁ 之前松开）：
- 按下后经时间 t₀ 匀加速至 v₁（加速度 a₀）
- 维持 v₁ 至 t₁
- 再经时间 t₀ 匀减速（加速度 a₀）至垂直速度为零
- $$v_1 = \frac{h_1}{t_1}$$
- $$a_0 = \frac{v_1}{t_0} = \frac{h_1}{t_1 t_0}$$

**长按**（超过 t₂ 松开）：
- 超过 t₁ 后 → 以固定加速度 a₂ 减速
- 松开后 → 以加速度 a₀ 匀减速至垂直速度为零 → 下坠

### 冲刺

- 分为三个阶段：**无敌冲刺** → **后摇**
- 冲刺过程中不受任何伤害，不可被打断
- 全程匀速，遇障碍物即停止（不会贴墙冲）
- 整个冲刺过程（含后摇）不被其他输入打断
- 冲刺开始前重置角色速度，以当前面朝水平方向为冲刺方向
- **允许空中冲刺**
- **冲刺距离**：3.5m（带跳跃可达 9.5m）
- **最远跳跃距离**：7m（不带冲刺）
- 各阶段持续时长待定（可能会有枪械/视角上下摆动动画表示前后摇，预计程序控制）

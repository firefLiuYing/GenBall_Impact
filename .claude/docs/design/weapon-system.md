# 武器系统

主角始终持有一把远程射击武器，随游戏进程不断进化和解锁新能力。武器分两个维度：**物理侧**（常规战斗通解）和**能力侧**（解密+特殊移动+战斗策略）。

---

## 物理侧：枪械进化

### 基础枪械模板

从一把基本的全自动矿工自卫步枪开始。后续可通过进化阶段切换为不同类型（机关枪、突击步枪、机炮、手枪、冲锋枪、榴弹炮等）。

**基础参数**（常规全自动步枪默认值）：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 射速 | 600 RPM | 每分钟射出子弹数 |
| 备弹量 | 30发 | 单个弹匣 |
| 换弹时间 | 2s | |
| 子弹伤害 | 待定 | |
| 子弹速度 | 待定 | 实体检测（非射线），需子弹预制体 |
| 散布（静止） | 待定 | 圆锥随机散布 |
| 散布（跑动） | 待定 | 大于静止散布 |
| 垂直后座力 | 待定 | 采用序列数组或公式计算两种方案 |
| 水平后座力 | 待定 | 同上 |
| 枪口归位 | 待定 | 松开开火键后是否自动归位 |

### 进化机制

#### 累计规则

- 周围一定范围内的怪物死亡 → 产生数据被枪械收集 → 击杀计数 +1
- 范围足够大（基本可见范围内的怪物死亡都会计入），实现上直接让怪物死亡时增加计数即可
- 击杀计数满足阶段阈值时 → UI提示按 **Q 键**手动进化
- 进化后击杀计数**不减少**，继续累积到下一阶段

#### 重置规则

- 在存档点存档后 → 击杀计数**全部清零** → 进化阶段**重置**
- 枪械阶段不允许回退，除非存档重置

#### 阶段解锁

- 初始只有第一阶段
- 第二阶段：首次到达存档点 + 点出第一个技能树节点后解锁
- 后续阶段：每累计解锁 N 个技能树节点后自动开启（具体 N 待定）

#### 进化演出

- 短时缓 + 枪械展开冲击波效果 + 枪械外观变化，尽量做得帅

### 进化阶段配置

每个阶段配置：**1 枪械基础形态 + N 个强化模块**

| 阶段 | 负载上限 | 模块槽数上限 | 解锁条件 |
|------|---------|-------------|---------|
| 阶段一 | 7 | 2 | 初始 |
| 阶段二 | 15 | 3 | 首个技能树节点 |
| 阶段三 | 25 | 3 | 累计解锁N个节点 |
| 阶段四 | 40 | 4 | 累计解锁N个节点 |

#### 阶段进化选项（草稿，部分待补）

**阶段一选项**：
- 射速加快，弹匣总量增加
- 射速变慢，单发伤害增加
- 击杀敌人时获得临时生命，可超过最大生命上限
- 之后阶段需要的击杀数减半

**阶段二选项**：
- 每发子弹命中时有一定概率造成范围AOE伤害
- 子弹穿透敌人且会弹射
- 每15秒获得一枚特殊榴弹，发射爆炸伤害
- 每击杀一只怪物，射速获得一定提升（有上限）
- 每击杀一只怪物，伤害获得一定提升（有上限）
- 枪械转变为霰弹枪，持枪移速提升
- 枪械转变为激光枪，需开镜瞄准，可穿透，蓄力提升伤害

**阶段三选项**（未写完）：
- 连续三次击中不同类型敌人，接下来三次伤害提升100%

**阶段四选项**（未写完）：
- 子弹不再造成伤害，命中三次后敌人立刻死亡 / 命中五秒后立刻死亡
- 枪械转变为光剑：左键伤害高，右键更高伤害，覆盖之前所有buff改为增加生命和移速
- 当前弹匣为空时，不受奥比斯伤害

## 能力侧：特殊能力枪械

### 通用规则

- 通过**轮盘**从物理枪械切换到能力枪械
- 使用完毕后**自动切回**物理枪械，进入冷却
- 主动取消也算使用完毕，进入冷却
- 每把能力枪有**各自独立的冷却时间**
- **脱战状态下**：所有冷却立刻清零，使用后无冷却，可立即切换

### 匣纳之枪（Stack Gun）

核心能力：吸收奥比斯入弹匣，按栈顺序（LIFO）射出。

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 弹匣容量 | 2 | 后续可扩展 |
| 冷却 | 10s（战斗状态） | 切回物理枪械后开始计算 |
| 操作 | 右键吸入 / 左键射出 / T取消 | |

**规则**：
- 任何类型的奥比斯都可被吸入，只占一格弹匣
- 吸入时奥比斯**不死亡**，保留被吸入前血量
- 开始射出后不可再吸入，直到弹匣清空
- 发射的奥比斯在一段时间后才进行伤害计算
- 所有射出的奥比斯碰撞造成伤害 + 击飞效果（保证泛用性）
- UI显示当前弹匣中奥比斯种类和顺序

### 连理之枪（Grapple Gun）

核心能力：发射抓钩，二次确认缩短绳子。

- 命中目标后可选择是否缩短绳子
- **较轻目标**（小型奥比斯等）：将目标抓来并晕眩
- **较重目标**（大型奥比斯、墙壁）：自己飞过去
- 详细参数待补充

### 裁径之枪（Path Gun）

核心能力：沿枪方向划定一段圆形通道。

- 所有进入通道的奥比斯**不受控地**沿通道方向移动
- 主角在通道中：顺方向加速，逆方向减速
- 详细参数待补充

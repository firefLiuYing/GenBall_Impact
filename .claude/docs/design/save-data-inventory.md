# 存档数据清单

> 用于和策划对齐哪些数据需要持久化、什么时候写入磁盘。
> 更新：2026-06-21

---

## 一、设置项

| 数据 | 说明 | 写入时机 |
|------|------|---------|
| 声音音量 | 主音量、音效、背景音乐等 | |
| 键位映射 | 移动、跳跃、射击、冲刺、换弹、进化等 | |
| 鼠标灵敏度 | X/Y 轴灵敏度 | |
| 语言 | 如有 | |
| 其他通用设置 | | |

---

## 二、玩家 Build 配置

| 数据 | 说明 | 写入时机 |
|------|------|---------|
| 各阶段配装 | 每个进化阶段选了什么主武器形态 + 哪些模块 | |
| 当前装备的模块列表 | 从已拥有模块中选了哪些装上 | |
| 已拥有的模块/蓝图列表 | 所有获取过的模块 | |
| 技能树节点激活状态 | 哪些节点已解锁 | |
| 阶段上限解锁进度 | 2~4 阶段是否已解锁 | |

## 三、世界进度

| 数据 | 说明 | 写入时机 |
|------|------|---------|
| 各场景存档点解锁情况 | 哪些篝火已点燃 | |
| 最后休息的存档点 | 死亡后从这里重生 | |
| 当前场景及存档点位置 | 读档后恢复到哪个场景哪个位置 | |
| 圆形数据数量 | 休息时由击杀计数转换而来 | |
| 失落数据数量 | 探索中获得 | |
| 第三货币数量 | 待定 | |
| 游戏总时长 | | |
| 角色属性等级 | | |

---

## 四、场景内一次性状态

| 数据 | 说明 | 写入时机 |
|------|------|---------|
| 敌人存活状态 | 大部分怪永久死亡不刷新；少数可刷新怪需标记 | |
| 可刷新敌人的标记 | 用于机关相关、必须刷新才能通过的敌人 | |
| 一次性机关状态 | 已触发的拉杆、已破坏的墙、已开启的密码锁等 | |
| 宝箱开启状态 | 已开过的宝箱不再出现 | |
| 拾取物收集状态 | 档案、道具等，已捡过的不再出现 | |
| 门/栅栏状态 | 酸蚀、破坏等，已打开的不再关闭 | |

## 六、运行时战斗数据（不写入磁盘）

| 数据 | 说明 | 休息时怎么处理 |
|------|------|--------------|
| 当前 HP | | 回满 |
| 当前护甲 | | 回满 |
| 当前弹匣余量 | | 回满 |
| 击杀计数 | 怪物死亡时累计 | 转换为圆形数据，然后清零 |
| 当前进化阶段 | | 重置到阶段一 |
| 能力枪冷却 | | 脱战即清零 |
| 匣纳之枪弹匣状态 | | 清空 |
| 当前激活的临时 Buff | | 清除 |

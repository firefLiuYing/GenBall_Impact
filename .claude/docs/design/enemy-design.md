# 敌人设计

## 敌人概览：奥比斯（Orbis）

奥比斯是球状能量生命体，是"圆"武器使用后世界的主要生命形式。基础数值仅以生命值为参考，以蓝色奥比斯为数值基准模板。

## 奥比斯种类

### 蓝色奥比斯（基础型）

- 最普通的奥比斯
- 伤害类型：碰撞造成伤害
- 作为数值基准模板

### 橙黄奥比斯（自爆型）

- 体型略小，更加愤怒
- 靠近到一定范围内立刻自爆
- 需要玩家识别并保持距离/远程处理

### 飞行奥比斯

- 飞行移动方式
- 飞行撞击造成伤害
- 对纯近战策略形成压力

### 吞噬者奥比斯

- 对角色进行捕食攻击
- 扣除一定血量后将角色朝一个方向喷出
- 影响玩家站位

### 相溶奥比斯

- 死亡时为周围所有奥比斯和主角恢复大量生命
- 击杀时机和顺序的策略选择：先杀它会奶活其他怪，后杀它则可以白嫖回血

### 酸液奥比斯

- 移动过的地面产生酸液，踩上扣血
- 自身也会喷吐酸液
- 区域控制型敌人，限制玩家走位
- 其酸液可用于溶解特定材质的墙壁/门/栅栏（机关交互）

## AI 行为

### 三种状态

#### 游荡巡逻
- 行为模式（择一）：
  - 以刷新点为中心，一定半径圆形内随机移动
  - 在两个点之间匀速来回巡逻
  - 原地不动
- **游荡探查半径**：玩家进入后立刻切换到战斗状态

#### 战斗状态
- **近战攻击型**：始终尽可能靠近玩家并发起攻击
- **远程徘徊型**：
  - 额外具有徘徊距离参数
  - 距离超过徘徊距离 → 尝试靠近
  - 距离小于徘徊距离 → 尝试远离
- 攻击优先级高于移动
- **战斗探查半径**（大于游荡探查半径）：玩家脱离后原地停滞一小段时间 → 进入脱离仇恨状态
- 若停滞期间玩家重回战斗探查半径 → 立刻恢复战斗状态

#### 脱离仇恨状态
- 持续移动返回刷新点
- 使用游荡探查半径

### 状态转换图

```
游荡巡逻 →（玩家进入游荡探查半径）→ 战斗状态
战斗状态 →（玩家脱离战斗探查半径+停滞）→ 脱离仇恨
脱离仇恨 →（玩家进入游荡探查半径）→ 战斗状态
脱离仇恨 →（到达刷新点）→ 游荡巡逻
```

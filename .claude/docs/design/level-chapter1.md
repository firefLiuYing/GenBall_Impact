# 关卡设计：第一章

## 概要

- **场景**：地下洞窟 + 废弃实验室交替
- **结构**：蚁穴式，自上而下探索
- **风格**：相对一本道/线性，不会让人记忆混乱
- **难度定位**：相对简单，侧重探索乐趣和解密式战斗

## 关卡流程（三段落）

### 第一段：苏醒与追寻光芒

- 从洞窟底部冬眠舱苏醒
- 在实验室房间门口被"小右"叫住 → 获得AI伙伴
- 在小右指引下充能激活战斗功能
- 主题：**向上移动**，追寻光芒试图回到地表
- 终点揭示：光芒来自核动力电灯泡（希望是假的）

### 第二段：实验室探索

- 转向另一条路，进入废弃实验室区域
- 地质演变导致部分实验室被泥石分隔
- 中间穿插洞窟段落（反复进出）
- 获取登顶电梯的信息
- 主题：**信息收集 + 横向探索**

### 第三段：登顶追逐

- 意外遭遇大量奥比斯追杀 → 追逐战
- 途中结合水下段落：利用匣纳之枪吸收水底奥比斯射击背后怪物，用后座力加速逃逸
- 最终利用岩浆冲力冲上地表
- 跨越时间性质，到达下一大关卡

## 机关类型

### 按交互方式分类

**简单交互**（一定范围内按F）：
- 拉杆开门
- 存档点交互
- 获取档案/道具
- 打开宝箱

**触发式**（自动触发）：
- 走到特定位置桥断掉
- 走到某处刷怪
- 触发剧情事件

**条件判定型**：
- 爆炸后可炸碎的墙壁
- 酸液可溶蚀的金属门/墙壁/栅栏/锁链
- 下落到一定速度才能抬起的跷跷板

**解密型**（需要特定操作）：
- 密码锁（按下F进入解锁界面，输入密码）
- 能量吸收装置（吸收一定能量奥比斯后启动 → 开门/传送/其他）

### 第一章出现的机关

| 机关 | 位置 |
|------|------|
| 酸蚀材质（金属墙/门/栅栏） | 洞窟+实验室 |
| 重量/冲击/爆炸破坏的地面/墙壁 | 洞窟 |
| 可推动的巨石/容器 | 洞窟 |
| 能量吸收仪器 | 实验室为主 |
| 可溶蚀金属锁链 | 实验室 |

## 关卡比例尺标准

### 角色与移动

| 参数 | 值 |
|------|-----|
| 人物高度 | 1.8m |
| 视角高度 | ~1.7m |
| 移动速度 | 5 m/s |
| 最大跳跃高度 | 3m |
| 冲刺距离 | 3.5m |
| 最远跳跃距离（冲刺+跳跃） | 9.5m |
| 最远普通跳跃距离 | 7m |

### 通道标准

| 类型 | 尺寸 | 说明 |
|------|------|------|
| 常规通道 | 高 2.2m × 宽 1.5m | 正常通行 |
| 不可通过矮通道 | 高 ≤ 1.6m | 需要特殊方式或绕路 |

### 跳跃平台设计标准

| 规则 | 值 |
|------|-----|
| 可跳过间距 | ≤ 最远跳跃距离 × 0.9 |
| 不可跳过间距 | ≥ 最远跳跃距离 × 1.3 |
| 可跳上高度 | ≤ 最大跳跃高度 × 0.8 |
| 不可跳上高度 | ≥ 最大跳跃高度 × 1.4 |

### 奥比斯通道尺寸

| 奥比斯体型 | 可通过的最小通道 |
|-----------|----------------|
| 常规（0.6m） | 较小 |
| 中型（1.3m） | 中等 |
| 大型（2.5m） | 常规通道及以上 |

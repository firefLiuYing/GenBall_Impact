# 游戏全貌概要

## 这是什么游戏

**第一人称软解密式射击战斗冒险**。核心体验是：利用环境机制和武器能力组合，自己发现每场战斗的"聪明解法"，通过内在动机获得成就感和爽快感。

## 核心循环

```
探索地下洞窟/实验室 → 遭遇奥比斯（敌人）→ 利用环境机制+枪械能力战斗
→ 击杀获得计数 → 到达存档点 → 计数转换为货币 → 技能树升级
→ 解锁新武器阶段/模块 → 继续探索更深层
```

## 核心设计原则

### 软解密式战斗
- **降低策略信息获取门槛**：通过教学和关卡设计建立思维路径（如"见到爆桶 → 远程射爆"），通过环境信息提前暗示敌人配置
- **保证底线体验**：即使没找到最优解，也能通过基础射击/移动操作通关，不卡关也不太无聊
- **内在动机驱动**：不过度依赖外部奖励，让"发现策略并执行成功"本身就是奖励

### 自由意志
- 物理侧枪械进化有分支选择，允许玩家自定义BD
- 战斗可绕过、可正面刚、可利用机制取巧 — 多种路径
- 地图中大部分敌人可打可不打

# 世界观与设定

## 世界观背景

### 设定概要

主角所处时代的人类文明已因"圆武器"战争而毁灭，时间过去了数百万年。所有灵长类生命体都已被"球"完全同化，人类文明的痕迹已从世界上消失。

### 关键事件链

1. **主角坠落**：主角在自己时代跌入深层奥比斯洞窟
2. **搜救与发现**：大量搜救队在搜救过程中意外发现地下丰富的圆石矿藏和源初性奥比斯
3. **秘密研究**：上层国家机关宣布该区域为生命禁区，实则偷偷在附近山脉地下建立实验室展开研究，尝试研发武器
4. **政体争斗**：各政体之间发生争斗，实验设施被废弃摧毁。包括对主角身体的研究也被永久封存于地下
5. **漫长沉睡**：主角从冬眠舱中醒来时，已是数百万年之后。山脉下沉，地壳变动

## 第一章设定：「未来和过去的洞窟」

### 场景

地下洞窟 + 废弃实验室交替出现，整体为蚁穴式结构，自上而下探索。洞窟原本是地面山峰，因下方奥比斯物质侵蚀而下沉形成。曾有冒险者踏足，留下人类尸骨和破坏痕迹。

### 剧情大纲

**三段流程**：

1. **苏醒与追寻光芒**
   - 从洞窟底部冬眠舱苏醒
   - 在即将出门时，听到背后传来声音"喂，看看后边儿！"
   - 捡到AI伙伴"小右"，在小右指引下激活战斗功能
   - 追寻光芒向上移动，尝试回到地表
   - 到达光芒出口时，发现那只是一个核动力电灯泡（假的希望）

2. **实验室探索**
   - 转向另一条路，进入被泥石分隔的旧实验室遗迹
   - 中间穿插洞窟段落
   - 获取关于过去实验的信息

3. **登顶追逐**
   - 获取登顶电梯的信息
   - 意外引发大量奥比斯追杀，陷入追逐战
   - 最终利用岩浆冲力冲上地表，跨越时间性质到达下一大关卡

### 第一章敌人配置

普通奥比斯、酸液奥比斯、自爆奥比斯、吞噬奥比斯（共四种）

### 第一章进度

- 养成树节点预计推进到第三阶段
- BOSS：大概率只做特殊战斗，不设传统BOSS

## 关键设定元素

- **奥比斯（Orbis）**：球状生物，与"圆"武器同源的能量生命体，设定上是世界文明毁灭后的主要生命形式
- **小右**：AI伙伴，嵌入主角右手的智能体，提供战斗功能和指引
- **圆石/圆武器**：与奥比斯同源的能量物质，曾作为武器毁灭了人类文明
- **源初性奥比斯**：更深层的原始奥比斯，待展开
